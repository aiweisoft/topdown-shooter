# Top-Down Shooter

A top-down perspective shooting game built with **Python + Pygame**. Playable on desktop (Windows).

## Features

- WASD / Arrow keys movement, mouse aim and shoot
- Triple spread shot + piercing bullets
- Wave-based level system (enemies get faster, more numerous each wave)
- Synthesized sound effects (shoot, explode, hit, level up)
- Particle explosion effects
- Game over / restart (R key)

## How to Play

| Action | Control |
|--------|---------|
| Move | W / A / S / D or Arrow Keys |
| Shoot | Hold Left Mouse Button |
| Restart | R |
| Quit | ESC |

## Getting Started

```bash
# Install dependencies
pip install pygame

# Run the game
python main.py
```

Or double-click `start.bat` on Windows.
